First use
1) Run create_workspace.bat

Generate plots
1) Define your plot pages in plot_definitions.py. Follow the example formatting. Available parameters can be found in 'ULG csv parameters.csv'
2) Place your .ulg files in the 'ULG Log Files' folder. If this folder doesn't exist, run create_workspace.bat
3) Your plots will be generated and saved into the Plots folder
4) A copy of the plots along with the csv timeseries data will be saved as a zipped folder in the Archive folder
